Tweet-a-lyze

Tweet-a-lyze helps analyze tweets. Live & Big.

It is a tweet search and analysis tool which scraps JSON data using the Loklak API



Why Loklak ?

Loklak is a server application which is able to collect messages from various sources, including Twitter. This server contains a search index and a peer-to-peer index sharing interface.

If one likes to be anonymous when searching things, want to archive Tweets or messages about specific topics and if you are looking for a tool to create statistics about Tweet topics, then Loklak is the best option to consider.

Easy to use

Collect and store a very, very large amount of Tweets and similar messages. Create one's own search engine for Tweets

Visualize & Evaluate Social Data

Create one's own Tweet search portal or statistical evaluations. Analyze large amounts of Tweets as a source for statistical data.

Anonymous Search

Omit authentication enforcement for API requests on the Twitter platform. Search anonymously on one's own search portal

If one wants to create an alternative Twitter search portal, the only way would be to use the official Twitter API to retrieve Tweets. But that interface needs an OAuth account and it makes the search portal completely dependent on Twitter's goodwill.

The alternative is, to scrape the Tweets from the Twitter HTML search result pages, but Twitter may still lock out on that IP address. To circumvent this, one needs many clients accessing Twitter to scrape search results. This makes it necessary to create a distributed peer-to-peer network of Twitter scrapers which can all organize, store and index Tweets. This solution was created with Loklak.

Best of all : A very generic to integrate with different microblogging services was developed, so this may be the incubator for an independent short message or Twitter-like platform.
